# Resource Mkmini

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
所有页面均为静态内容，不依赖后端服务，可直接通过浏览器打开或部署到任意静态托管平台。

## 项目简介

本仓库旨在集中存放各类独立的 HTML 页面资源。  
每个页面可以是一个小工具、信息展示页、交互原型或其他用途的网页。  
项目不针对任何特定域名或网站，仅作为页面归档与发布的中转站。

## 目录说明

- `pages/` — 存放所有 HTML 页面文件，每个文件为一个独立页面
- `assets/` — 公共资源目录（如 CSS、JavaScript、图片等），供页面引用
- `docs/` — 可选文档或说明文件

页面命名建议使用简洁的英文或拼音，便于识别和管理。

## 页面归档说明

- 每个 HTML 页面应自包含其样式与脚本，或引用 `assets/` 下的公共资源
- 页面之间相互独立，不依赖本仓库外的任何外部资源（除非必要）
- 如需引入外部库（如字体、图标），建议使用 CDN 链接并在页面内注明来源

## 维护说明

- 欢迎提交 Pull Request 添加新的页面或改进现有页面
- 请确保页面内容合法合规，不包含恶意代码或侵权内容
- 仓库维护者会定期检查页面可用性，但不对页面内容负责
- 若发现页面无法正常显示或存在安全问题，请提 Issue 反馈

## 使用方式

1. 克隆或下载本仓库
2. 在浏览器中直接打开 `pages/` 下的 HTML 文件
3. 或将整个仓库部署到静态托管服务（如 GitHub Pages、Netlify 等）

## 许可

本仓库中的代码及资源默认采用 MIT 许可协议，具体页面如有特殊声明，以页面内标注为准。
